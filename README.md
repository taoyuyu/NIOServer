# [NIOServer](https://github.com/taoyuyu/NIOServer)

## What is NIOServer
* IM(Instant Messaging)
* Web Application
* Push Notification
* Reverse Proxy

## Getting Started
### Prerequisites
* **JDK**  
Oracle JDK 7 and above  
Open JDK 7 and above  
IBM JDK 7 and above
* **OS**  
Linux (any flavor)  
Mac OSX (10.6 and up)  
Windows (XP and up)  
Windows Server (2008 and up)  
Oracle Solaris

### Installation
**Build from the source:**   
> mvn install
